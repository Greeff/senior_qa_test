How to setup Behat

1. PHP 5.6 and above
2. You need to have composer package in order to install Behat
- https://getcomposer.org/download/
3. Install Behat
- $ php composer.phar install
4. Then run Behat
- $ vendor/bin/behat
5. Or you can run specific scenario with
- $ vendor/bin/behat features/user_login.feature
