Script is written in Python3 Running on Firefox Webdriver

————————————————————————————————————————————————————————

Requirements to Run Test

-Python3
brew Install Python3

-Unittest
pip3 install Unittest

-Selenium Webdriver
pip3 install selenium

-Firefox Browser Version 54.0.1 and above

-Firefox Gecko Driver
Install Gecko Driver and ensure that this is in the root directory: 
PATH="/Users/<your-name>/Downloads/geckodriver:$PATH"
export PATH

To Run Test open Terminal: 

1) Locate cloudcurrency.py within terminal
2) type python3 cloudcurrency.p enter. 

Expected: 
Two Firefox Windows should fire up, the second window should show a demo of the test.
